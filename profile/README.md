# Goaly

**No-loss football prediction on Arbitrum.** Stake USDT on match outcomes — your principal is never at risk; only the yield it earns funds the prizes.

Every player gets a self-custodial [Tether WDK](https://wdk.tether.io) wallet — keys on the device, no seed phrase — to stake and get paid in USDT. Stakes pool into an ERC-4626 vault and earn yield on [Morpho](https://morpho.org); the principal stays fully redeemable, and winners split a prize funded entirely by that yield. An autonomous agent, running on its own WDK wallet, manages the pool. Built for the **Tether Developers Cup** — WDK (Wallets) track.
